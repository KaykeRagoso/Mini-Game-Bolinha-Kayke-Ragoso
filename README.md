# Mini-Game-Bolinha---Kayke-Ragoso
Este projeto é um mini game feito em HTML5 e Canvas, onde o jogador interage com uma bolinha vermelha usando o mouse. O objetivo é explorar conceitos de física básica, como gravidade, elasticidade e atrito, enquanto aprende manipulação de canvas, eventos de mouse e animação frame a frame.

## O jogo permite:

- Segurar e arrastar a bolinha com o mouse.

- Soltar para lançar a bolinha em qualquer direção.

- Observar a bolinha quicando nas bordas e reagindo realisticamente à física.

## Tecnologias Utilizadas

- HTML5 – estrutura da página.

- CSS3 – estilo, centralização e cursor interativo.

- JavaScript – lógica de física, colisões e eventos de mouse.

- Canvas API – renderização e animação da bolinha.

## Mecânicas do Jogo

- Gravidade – a bolinha é constantemente puxada para baixo.

- Elasticidade (Bounce) – quica nas bordas da tela mantendo velocidade proporcional.

- Atrito – reduz a velocidade horizontal após colisões, tornando o movimento natural.

- Controle pelo mouse – clicar e arrastar a bolinha para lançá-la.

- Linha de força – mostra direção e intensidade do lançamento enquanto arrasta.

## Como Jogar

- Abra o arquivo index.html no navegador.

- Clique na bolinha e segure o mouse.

- Arraste na direção desejada para lançar.

- Solte o mouse e veja a bolinha voar e quicar nas bordas.
